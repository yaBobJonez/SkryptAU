SkryptAU

SkryptAU (working title "Simple Transliterators") is a Free program comprised of a set of simple (utilizing char-by-char iteration with lookup) transcribing functions that implement my concepts of alternative writing systems (alphabetic) for several common languages.

Languages canonically written using the Latin script are converted to using Cyrillic, while a few Cyrillic-based languages are rewritten in Latin. SkryptAU does not adhere to any official transliteration standards and is based solely on my own ideas, while offering a few options to customize its behaviour. Orthographic exceptions handling is provides for French (extensive) and Spanish (a few common words). Licensed under MPL 2.0; you are free to contribute ;)

Such name has been chosen because "skrypt" looks like a distorted word "script" (writing system) and the original language was Polish, while AU stands for "alternate universe".

Featured languages:

  • Cyrillic Polish
  • Cyrillic French
  • Cyrillic German
  • Cyrillic Spanish
  • Cyrillic Italian
  • [WIP] Cyrillic Dutch
  • Latin Ukrainian
  • [WIP] Latin Belarusian
  • [WIP] Latin Korean
